# Agent Rules

These rules apply to Lyra, Codex, Kiro, and other agents using this repo.

## Before starting work

1. Pull latest git changes.
2. Read:
   - `01-Knowledge/Context/Preferences.md`
   - `01-Knowledge/Context/Environment.md`
   - `01-Knowledge/Context/Active Work.md`
   - `01-Knowledge/Context/Vault Operating Rules.md`
   - relevant project overview/index files under `02-Projects/`

## During work

- Prefer investigation before implementation.
- Do not guess project context if notes exist.
- Do not write secrets to this repo.
- Keep notes distilled, not raw chat dumps.
- Link related notes using Obsidian `[[wikilinks]]` where useful.

## After work

- Update relevant project notes if durable facts changed.
- Add a decision record for major choices.
- Update `Active Work.md` if high-level status changed.
- Commit and push changes.
