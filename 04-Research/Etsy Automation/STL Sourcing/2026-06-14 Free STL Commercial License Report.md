# Etsy STL Report — Free Files Only

Research date: 2026-06-14  
Scope: only **free-download** STL/OBJ/3MF files. Paid files, paid memberships, and commercial Patreon-only options are excluded from candidate recommendations.  
Goal: identify files Hammad could potentially use for an Etsy 3D-print store, with commercial-use/license vetting.

## Key rules used

- **Free does not mean commercial-use allowed.** The report separates price from license.
- **File license does not clear franchise IP.** CC BY/CC0 may allow sale of printed objects under the designer’s file license, but Marvel/DC/Disney/Sony/anime/game rights can still create Etsy takedown risk.
- **Commercial-use OK file licenses:** CC0/Public Domain, CC BY, CC BY-SA, CC BY-ND, GPL-like/open licenses, or explicit text allowing sale of physical prints.
- **Rejected for Etsy print sales:** CC BY-NC, CC BY-NC-SA, CC BY-NC-ND, Cults PU/private use, MakerWorld Standard Digital File License, personal-use-only text, unclear reuploads/provenance.
- **Attribution:** For CC BY / BY-SA / BY-ND, include designer name, model title, source URL, and license in the Etsy listing or packaging slip.

# Free-only recommendation

## Best items to actually start with

1. **Cosplay helmet wall mount** — generic, free, CC BY/CC0, low IP risk.
2. **Universal helmet stand** — generic, free, CC BY/CC0, low IP risk.
3. **Cybercore Gauntlet** — free/public-domain style; only use generic sci-fi branding.

## Items with usable file licenses but serious marketplace risk

- Red Hood, Batman, Iron Man, Doctor Doom, Deadpool, Wolverine, Chainsaw Man, Naruto, Attack on Titan, Helldivers, ODST/Halo.
- For these, the STL license may allow physical prints, but the underlying franchise designs can still be removed from Etsy.

## Items not cleanly usable from free-only search

- **Wolverine-style raw cowl:** no fully clean free commercial-safe candidate found because of conflicting terms / Cults PU / provenance issues.
- **Tokyo Ghoul/Kaneki mask:** no clean free commercial-safe candidate found; best free candidate has reupload/provenance concerns.
- **Helldivers helmet:** free files exist, but licenses/provenance are unclear and game IP risk is high.

## Suggested next step

Use the generic free files for early testing/listings. For the high-demand character props, either:

- commission original “inspired-by but not matching” files with full commercial rights, or
- obtain direct written permission from the STL designer and still avoid official franchise names/logos in Etsy listings.
