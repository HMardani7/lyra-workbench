## Durable conclusions

- Free does **not** mean commercial-use allowed.
- Commercial-friendly file licenses still do **not** clear Marvel/DC/Disney/Sony/anime/game IP.
- Best low-risk starting products are generic helmet mounts and helmet stands.
- Fan-art helmets/masks can have file-level permission but still carry Etsy takedown risk.
- For CC BY / CC BY-SA / CC BY-ND candidates, attribution should be included in listings or packaging notes.
