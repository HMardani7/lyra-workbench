# Closer — Overview

## Purpose

Closer is a long-distance relationship app MVP focused on helping couples feel emotionally present through lightweight rituals, quick signals, communication prompts, and shared availability.

## Canonical repo

- Path: `/srv/hermes-repos/closer`
- GitHub: `HMardani7/closer`

## Current status

The repository is an early monorepo scaffold. Frontend is intentionally empty except placeholder structure; backend is a minimal FastAPI-style scaffold with no business features implemented yet.

## MVP scope

In MVP:

- authentication and onboarding
- partner linking via invite/code
- shared couple home screen
- quick emotional signals
- shared availability / calendar-style scheduling
- daily couple question mechanic
- basic settings and profile

Not in MVP:

- widgets
- whiteboard
- AI smart scheduling
- memories/scrapbook
- advanced analytics
- monetization/payments
